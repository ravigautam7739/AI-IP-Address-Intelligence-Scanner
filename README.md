# 🌐AI IP Address Intelligence Scanner

A Python-based **AI-style IP intelligence scanner simulator** that analyzes an IP address and generates a detailed network security report.

This project creates a cybersecurity terminal experience inspired by AI assistants like JARVIS.

---

# 🚀 Features

✔ IP address input system
✔ AI-style scanning animation
✔ ISP information simulation
✔ ASN information analysis simulation
✔ VPN detection simulation
✔ Proxy detection simulation
✔ Tor exit node checking simulation
✔ Blacklist status simulation
✔ Network reputation score
✔ Cybersecurity terminal interface

# 🧠 How It Works

1. User enters an IP address

Example:

```text
Enter IP Address:
8.8.8.8
```

2. Scanner starts:

```text
Initializing Intelligence Engine

Locating IP Address

Checking ISP Database

Checking ASN Information

Scanning Abuse Reports

Checking VPN Detection

Checking Proxy Detection

Checking Tor Exit Nodes

Analyzing Network Reputation

Searching Blacklist Database

Running AI Threat Analysis
```

3. A final IP security report is generated.
